# Phase 9 Plan 1: AI Document Extraction and Signature Routing

## Objective
Build document upload with AI extraction via Gemini, plus signature status workflow tracking visible in entity documents tab.

## Context
- @src/app/api/ai/draft/route.ts — Gemini pattern from Phase 8
- @src/components/entities/entity-detail-tabs.tsx — Documents tab placeholder
- @src/lib/db/types.ts — Document type with extracted_data, signature_status

## Tasks

### Task 1: Document upload, AI extraction API, and extraction display
type="auto"

Create `/api/ai/extract` POST route:
- Accepts document text content and entity ID
- Uses Gemini to extract: parties, key dates, obligations, governing law
- Returns structured JSON extraction result
- Template fallback with sample extracted data when no API key

Create document upload UI in documents tab:
- Upload area (text/paste for demo — no real file storage needed)
- Trigger extraction button
- Display extracted data in readable summary cards (parties, dates, obligations, governing law)
- Store extraction results in component state

Done criteria:
- User can paste/upload document text
- AI extraction returns structured JSON
- Extracted data displayed in readable format

### Task 2: Signature status workflow and status tracker
type="auto"

Build signature routing workflow:
- Document status state: draft -> sent_for_signature -> signed
- Status change buttons/controls
- Step indicator component showing current position in workflow
- Status persists in component state (demo-level persistence)
- Visual status tracker (3-step indicator with icons)

Update documents tab to show:
- List of documents with status badges
- Click to see extraction details
- Status workflow controls

Done criteria:
- Status workflow: draft -> sent -> signed with visual step indicator
- Status changes persist and display correctly
- Documents tab shows full document list with status

## Verification
- Upload text triggers AI extraction with structured output
- Extracted data displays in readable summary
- Status workflow progresses through 3 states
- Step indicator reflects current status
