2. Add a custom filter function for array-based faceted filtering. Define it outside the component:
```typescript
const facetedFilter: FilterFn<EntityHealthSummary> = (row, columnId, filterValue: string[]) => {
  if (!filterValue?.length) return true
  return filterValue.includes(String(row.getValue(columnId)))
}
```

3. Update the columns to use this filter function — either:
   - Pass it via `useReactTable({ filterFns: { faceted: facetedFilter } })` and set `filterFn: 'faceted'` on the relevant columns in columns.tsx
   - OR: set `filterFn` directly on the column defs in columns.tsx — choose whichever is cleaner. The faceted filter must work for: `country_name`, `status`, `risk_level`, `entity_type`.

4. Add the global filter function for name search:
```typescript
const globalFilterFn: FilterFn<EntityHealthSummary> = (row, _columnId, filterValue: string) => {
  const search = filterValue.toLowerCase()
  const name = row.original.name.toLowerCase()
  const legalName = row.original.legal_name.toLowerCase()
  return name.includes(search) || legalName.includes(search)
}
```

5. Pass `globalFilterFn` to `useReactTable({ globalFilterFn })`.

6. Render `<DataTableToolbar table={table} />` above the `<Table>` component, with a small gap (`space-y-4`).

7. Update the entity count at the bottom to show filtered count: `{table.getFilteredRowModel().rows.length} of {data.length} entities`

8. If columns.tsx needs updating to add `filterFn: 'faceted'` on the four filterable columns, update it as well.
