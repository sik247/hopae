# Phase 10 Plan 1: Notion and Google Drive Integration Panels

## Objective
Add inline Notion and Google Drive panels to entity detail pages showing linked pages and documents with realistic demo data.

## Context
- @src/components/entities/entity-detail-tabs.tsx — Entity detail tabs
- @src/lib/db/types.ts — Entity types

## Tasks

### Task 1: Integration panels and updated entity detail tabs
type="auto"

Create Notion integration panel:
- Component showing linked Notion pages per entity
- Demo data: realistic page titles (e.g., "Board Minutes Q4 2025", "Entity Registration Notes")
- Each entry shows: page title, last edited date, external link icon
- "Connect Notion" placeholder when no data
- Uses realistic Notion-style formatting

Create Google Drive integration panel:
- Component showing linked Drive files per entity
- Demo data: realistic file entries (e.g., "Articles_of_Incorporation.pdf", "Annual_Return_2025.xlsx")
- Each entry shows: file name, file type icon, last modified date, external link
- "Connect Google Drive" placeholder when no data

Add "Integrations" tab to entity detail tabs:
- New tab alongside overview, compliance, documents, agreements
- Contains both Notion and Drive panels side by side
- Inline within entity profile (user doesn't leave platform)

Done criteria:
- Integrations tab visible in entity detail
- Notion panel shows linked pages with titles and dates
- Drive panel shows files with names, types, dates
- Both panels embedded inline
- Demo data looks realistic per entity

## Verification
- New Integrations tab appears in entity detail
- Notion panel displays page entries
- Drive panel displays file entries
- User stays on platform (no external navigation required)
